#  Recipe Search App

A simple recipe search web app built using HTML, CSS, and JavaScript.  
It fetches real-time recipe data from TheMealDB API and dynamically renders results.

---

##  Features

- Search recipes by ingredient or name
- Displays recipe image, name, and cuisine
- Shows step-by-step instructions
- Lists ingredients dynamically
- YouTube recipe link support
- Clean responsive UI (basic)

---

##  Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- TheMealDB API

---

##  API Used

- https://www.themealdb.com/api.php
